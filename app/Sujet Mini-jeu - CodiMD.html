  1. Rajouter une logique minimale dans le tread du jeu. Remarque, il est nécessaire de faire un lock sur le canvas pour pouvoir écrire dessus (multi-thread oblige)
@Override
public void run() {
    while (running) {
        canvas = null;

        try {
            canvas = this.surfaceHolder.lockCanvas();
            synchronized(surfaceHolder) {
                this.gameView.update();
                this.gameView.draw(canvas);
            }
        } catch (Exception e) {} 
        finally {
            if (canvas != null) {
                try {
                    surfaceHolder.unlockCanvasAndPost(canvas);
                } catch (Exception e) {
                    e.printStackTrace();
                }
            }
        }
    }
}

Création du jeu

Version initiale

Une balle est au centre de l’écran et se déplace régulièrement dans une des 4 directions (gauche haut, droite, bas). Lorsque vous cliquez sur l’écran, la balle change de direction et accélère. Le but du jeu est d’empécher la balle d’atteindre le bord de l’écran.

Lorsque la balle atteind le bord, le jeu se termine et affiche un score (le temps de survie en secondes).

Version finale à rendre

Chaque groupe doit proposer et implémenter un jeu se basant sur les briques précédentes et intégrant les 3 capteurs suivants:

  • Capteur de luminosité
  • Accélérometre
  • Touch screen

Vous devez aussi avoir au moins deux activités avec des informations qui passent entre les activités (en parametres). Un ecran de score par exemple

Vous devez aussi utiliser des sharedPreferences.

Enfin vous devrez utiliser un gestionnaire de version. Le but de cette partie est de tester l’intégration de plusieurs branches pour produire une seule application.

Vous trouverez ici https://linuxfr.org/news/tour-d-horizon-des-images-libres-et-pas-libres des liens vers des sites contenant des images libres de droit. Vous trouverez ici des icones en opensource : https://iconduck.com/

Si vous avez fini l’intégration des 3 capteurs et que votre jeu est fini, rajoutez les éléments suivants:
